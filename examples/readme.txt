This directory contains examples that illustrate different aspects of PRL:

grid_motion/
  Shows how to define variables and discrete (table) factors. 
  Also shows how to peform sum-product operations on factors.

graph/
  Shows how to define graph structures and invoke graph algorithms.

loopy_bp/
  A fully worked out example that lets the user run loopy belief 
  propagation on graphs with unary and binary potentials.

filter/
  Illustrates filtering in dynamic Bayesian networks.

ipf/
  Illustrates parameter learning with iterative proportional fitting.
